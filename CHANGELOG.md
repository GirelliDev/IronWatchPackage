# Changelog - IronWatchPackage (Uso Restrito)

Este changelog documenta as atualizações internas do sistema IronWatch.
Distribuição externa é proibida.

## [1.0.0] - 2025-11-03

### Adicionado

- Estrutura unificada do repositório (Servidor, Admin, Usuário Mobile e Desktop)
- Documentação técnica interna (`docs/`)
- Licença Proprietária Girelli Dev
- Configuração inicial para uso privado e controle de acesso

### Planejado

- Pipeline de build interno (CI/CD)
- Integração com servidor de deploy (Debian)
- Sincronização segura entre módulos via `shared/`
- Sistema de autenticação reforçada e logs avançados
